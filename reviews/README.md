# Review Workspace

## Purpose
Canonical location for review cycles and machine-readable findings.

## Structure
- `reviews/templates/` reusable templates
- `reviews/RV-YYYYMMDD-rNN/` one review cycle

## Active Cycle
- `RV-YYYYMMDD-rNN`

## Required Artifacts Per Cycle
- `findings.md`
- `findings.json`
- `gate_results.md`
- `reviewer_handoff.md`
- `snapshot_manifest.json`

## Canonical Vocabulary
Use status lifecycle and ID scheme from:
- `docs/agents/golden_repo_governance_proposal_v1.md`
